# Малахит

Локальное приложение для создания и управления сценариями видеоконтента.

## Возможности

- создание и редактирование сценариев;
- управление блоками сценария;
- работа с шаблонами;
- добавление заметок к строкам текста;
- расчет примерной длительности озвучивания;
- экспорт сценария в PDF.

## Запуск

Требуется Python 3.12 или новее.

```bash
pip install -r requirements.txt
python main.py
```

## Зависимости

Основные зависимости перечислены в `requirements.txt`.
